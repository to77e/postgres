# postgres
Курс «PostgreSQL для разработчиков приложений»

## Запуск PostgreSQL в Docker локально

```bash
make up
```

## Остановка PostgreSQL в Docker локально

```bash
make down
```

Кофигурация PostgreSQL указана в файле `docker-compose.yml`.

## Подключение к PostgreSQL

```bash
make psql
```
